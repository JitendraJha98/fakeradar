# Roadmap

Effort estimates assume one engineer, part-time, one A100.

## v0.1 — Credible baseline (3–4 weekends)
- [x] Modular pipeline: gradient + frequency + optional CLIP branch, gated fusion
- [x] Robustness-first training, native-res multi-crop inference
- [x] CLI, REST API, ONNX export, CI, tests
- [ ] Train `fast` tier on GenImage SD1.4-only protocol (A100, ~2–4h)
- [ ] Publish weights to HF Hub + cross-generator & robustness tables
- [ ] PyPI release

## v0.2 — Accuracy tier + demo (2–3 weekends)
- [ ] Train `accurate` tier (frozen CLIP ViT-B fusion)
- [ ] Temperature calibration on held-out val; report ECE
- [ ] Hugging Face Space demo (Gradio, `demo/app.py`)
- [ ] ONNX int8 quantization + CPU latency table

## v0.3 — In-the-wild robustness (4+ weekends)
- [ ] Add 2025–26 generators to eval: Flux, MJ v7, SD3.5, Imagen, GPT-4o images
- [ ] Community failure-case intake -> monthly eval refresh
- [ ] Public leaderboard page generated by `fakeradar benchmark`
- [ ] Explainability: patch-score heatmaps

## v0.5 — Attribution beta
- [ ] Multi-class head: which generator family made this?
- [ ] Browser-extension proof of concept (ONNX runtime web)

## v1.0 — Paper + stable API
- [ ] GRAFT paper (see paper/) with full experiments and ablations
- [ ] Semver-stable Python API, versioned model zoo
