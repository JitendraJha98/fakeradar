# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**TL;DR:** Be respectful. No harassment, personal attacks, or gatekeeping.
Assume good faith. Disagreements about detection methods are welcome;
disrespect toward people is not.

Report violations by opening a confidential issue or emailing the maintainer.
Maintainers may remove comments, commits, or contributors that violate this code.
